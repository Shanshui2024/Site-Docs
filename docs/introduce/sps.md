# 关于山水画路服务器
## 基础介绍
山水画路服务器是山水画路社区下设的一个公益服务器，服务器来源于[简幻欢](https://simpfun.cn/)的机器

您可以与好友体验生存的乐趣（多的我不想说了，自己看[官网](https://game.spr.us.kg/)）
